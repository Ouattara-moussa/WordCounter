# WordCounter

## Description
Programme qui compte le nombre total de mots dans un fichier texte.

## Fonctionnalités
- Lecture d'un fichier texte
- Comptage des mots (séparés par des espaces)
- Affichage du résultat

## Comment exécuter
1. Placer un fichier texte nommé `fichier.txt` à la racine du projet
2. Compiler : `javac Main.java`
3. Exécuter : `java Main`

## Tests effectués
- `fichier.txt` : 47 mots
- `fichier2.txt` : 11 mots
- `fichier3.txt` : 0 mots (fichier vide)
